## 📖 О проекте

**Sputnik** — это MVP интеллектуальной системы защиты космических аппаратов от угроз космической погоды: солнечного ветра, корональных выбросов массы (CME) и потоков заряженных частиц.

Система в реальном времени получает данные с серверов NOAA SWPC (National Oceanic and Atmospheric Administration — Space Weather Prediction Center), анализирует их с помощью обученной нейронной сети PyTorch, рассчитывает оптимальные параметры плазменно-электромагнитного щита и визуализирует защитную оболочку вокруг аппарата.

### Ключевые возможности

- Получение актуальных данных космической погоды от NOAA SWPC в реальном времени
- Предсказание уровня угрозы с помощью нейросетевой модели (PyTorch)
- Расчёт физических параметров магнитного поля и плотности плазмы
- Регулировка тока в катушках генератора защитного поля
- Прогнозирование времени до опасной зоны
- Визуализация плазменно-магнитной защитной оболочки в виде графиков

## 🔍 Описание файлов и кода

### `ai.py` — Главный модуль

Точка входа в программу. Объединяет все компоненты системы в единый пайплайн:

1. **Принимает ввод пользователя** — скорость аппарата и расстояние до зоны угрозы.
2. **Загружает предобученную модель** `solar_shield.pt` через `torch.load()`.
3. **Запрашивает данные космической погоды** через модуль `get_data.py`.
4. **Формирует входной тензор** из параметров пользователя и данных NOAA.
5. **Прогоняет данные через нейронную сеть** и получает прогноз уровня угрозы.
6. **Рассчитывает физические параметры защиты** (плотность плазмы, сила тока, напряжённость поля).
7. **Вызывает `graph.py`** для построения визуализации.
8. **Выводит результаты** в консоль.

**Пример потока данных:**
```
Ввод → get_data (NOAA API) → Тензор → Нейросеть → Параметры защиты → Визуализация
```

---

### `get_data.py` — Получение данных космической погоды

Отвечает за взаимодействие с внешним API NOAA SWPC.

**Что делает:**
- Отправляет HTTP-запросы к эндпоинтам NOAA SWPC JSON API
- Парсит данные о солнечном ветре: скорость, плотность частиц, температура
- Парсит данные о межпланетном магнитном поле (IMF, Bz-компонента)
- Обрабатывает возможные ошибки соединения и возвращает резервные значения
- Нормализует входные данные перед передачей в модель

**Используемые эндпоинты NOAA SWPC:**
- `https://services.swpc.noaa.gov/products/solar-wind/plasma-7-day.json` — данные солнечного ветра (плотность, скорость, температура)
- `https://services.swpc.noaa.gov/products/solar-wind/mag-7-day.json` — данные магнитного поля (Bx, By, Bz, Bt)

**Ключевые параметры, получаемые из API:**

| Параметр | Описание | Единицы |
|----------|----------|---------|
| Скорость солнечного ветра | Скорость потока заряженных частиц | км/с |
| Плотность частиц | Количество протонов в единице объёма | p/см³ |
| Температура плазмы | Температура солнечного ветра | К |
| Bz (IMF) | Z-компонента межпланетного магнитного поля | нТл |

### `graph.py` — Визуализация

Модуль построения графиков с использованием `matplotlib`.

**Что визуализирует:**
- Двумерную или трёхмерную проекцию плазменно-магнитной защитной оболочки
- Динамику изменения уровня угрозы во времени
- Распределение напряжённости поля вокруг аппарата
- Сравнение текущих параметров с пороговыми значениями

**Графики строятся в полярных или декартовых координатах** в зависимости от типа визуализации защитной оболочки.

---

### `solar_shield.pt` — Предобученная модель

Файл с весами нейронной сети в формате PyTorch (`.pt`). Загружается в `ai.py` через:

```python
model = torch.load('solar_shield.pt')
model.eval()
```

**Архитектура модели (предположительно):** полносвязная нейронная сеть (MLP — Multi-Layer Perceptron) с несколькими скрытыми слоями, обученная предсказывать оптимальные параметры защитного поля по данным космической погоды.

**Входные признаки модели:**
- Скорость аппарата
- Расстояние до зоны угрозы
- Данные солнечного ветра (скорость, плотность, температура)
- Компоненты IMF (Bz и другие)

**Выход модели:** числовое значение (уровень угрозы / параметры поля), например `1331.1876`

### `data.learn` — Датасет для обучения

Файл с обучающими данными для нейронной сети. Содержит исторические данные космической погоды и соответствующие параметры защитного поля. Используется при дообучении или переобучении модели.

**Формат:** текстовые или CSV-подобные строки с парами (входные параметры → ожидаемый выход).

## 🚀 Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/amirtime28/Sputnik.git
cd Sputnik
```

### 2. Убедитесь, что установлен Python 3.12

```bash
python3.12 --version
# Python 3.12.x
```

> Если Python 3.12 не установлен — скачайте его с [python.org](https://www.python.org/downloads/)

### 3. (Опционально) Создайте виртуальное окружение

```bash
python3.12 -m venv venv
source venv/bin/activate        # Linux / macOS
# venv\Scripts\activate         # Windows
```

### 4. Установите зависимости

```bash
python3.12 -m pip install numpy pandas matplotlib torch torchvision torchaudio requests
```

> Для систем с CUDA (GPU) рекомендуется установить PyTorch с поддержкой GPU — инструкции на [pytorch.org](https://pytorch.org/get-started/locally/)

## 🧠 Как работает ИИ

```
             ┌──────────────────────┐
             │    Ввод пользователя │
             │  (скорость, расстояние)│
             └──────────┬───────────┘
                        │
             ┌──────────▼───────────┐
             │    NOAA SWPC API     │
             │  (данные солн. ветра)│
             └──────────┬───────────┘
                        │
             ┌──────────▼───────────┐
             │  Формирование тензора│
             │  (нормализация данных)│
             └──────────┬───────────┘
                        │
             ┌──────────▼───────────┐
             │  Нейросеть (MLP)     │
             │  solar_shield.pt     │
             └──────────┬───────────┘
                        │
             ┌──────────▼───────────┐
             │  Расчёт параметров   │
             │  защитного поля      │
             └──────────┬───────────┘
                        │
             ┌──────────▼───────────┐
             │  Визуализация        │
             │  (graph.py)          │
             └──────────────────────┘
```

**Что анализирует ИИ:**
- Совокупный уровень геомагнитной угрозы по данным солнечного ветра
- Оптимальную плотность плазмы для создания защитного экрана
- Требуемую силу тока в катушках электромагнитного генератора
- Прогнозируемое время до вхождения в опасную зону
