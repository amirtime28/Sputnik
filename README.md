
# Интеллектуальная система плазменно-электромагнитной защиты космического аппарата

## Описание

Проект представляет собой MVP интеллектуальной системы, которая получает данные космической погоды (NOAA SWPC), анализирует уровень солнечной активности и рассчитывает параметры плазменно-электромагнитной защиты космического аппарата.

Система объединяет:
- физическую модель магнитного поля и плазмы  
- алгоритмы искусственного интеллекта  
- прогнозирование уровня угрозы  
- визуализацию защитной оболочки  

---

## Предназначение

Цель проекта — разработка адаптивной системы защиты космических аппаратов от солнечного ветра, корональных выбросов массы и потоков заряженных частиц.

Проект демонстрирует:
- интеграцию реальных данных космической погоды  
- работу ИИ-модуля  
- расчёт параметров магнитного поля и плазмы  
- построение графиков защитной оболочки

Требуется Python 3.12.

Установка зависимостей:

```
python3.12 -m pip install numpy pandas matplotlib torch torchvision torchaudio requests
```

## Запуск

1. Перейдите в папку проекта:

```
cd Sputnik
```

2. Запустите программу:

```
python3.12 ai.py
```

---

## Входные данные

Пользователь вводит:
- скорость космического аппарата  
- расстояние до потенциальной зоны угрозы  

Система автоматически получает данные космической погоды с NOAA SWPC.

---

## Работа ИИ

ИИ-модель:

- анализирует входные параметры  
- оценивает уровень угрозы  
- рассчитывает плотность плазмы  
- регулирует силу тока в катушке  
- прогнозирует время до опасной зоны  

---

## Пример работы

- прогноз динамики угрозы
